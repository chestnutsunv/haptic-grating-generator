# haptic-grating-generator
MATLAB tool to generate grating patterns and laser-cut PDFs for haptic experiments.
触覚弁別閾実験のために作成した，レーザーカッター用の白黒赤グレーティングPDF生成ツールです．
## Features
- Generate periodic and random grating images with adjustable parameters.
- Export laser-cut-ready PDF (vector/bitmap format suited for CO₂ laser cutters).
- Parameter presets for common materials and pitches.
